# Chronos TierZoo Narrative Pipeline

## Purpose

이 파이프라인은 역사 사건과 인물을 TierZoo 스타일의 게임 메타 분석 구조로 설명하기 위한 생성 규칙이다.

모든 콘텐츠는 다음 관점으로 작성한다.

역사 인물 = player build  
세력 = faction / guild  
왕조 = server  
시대 변화 = expansion patch  
쿠데타 = PvP encounter  
숙청 = hard counter event  


# Pipeline Overview

모든 대본은 반드시 아래 순서를 따른다.

1 Hook
2 Server Meta Setup
3 Base Stats Analysis
4 Special Abilities
5 Matchups
6 Meta Shift Analysis
7 Tier Placement Verdict


# Stage 1 Hook

반직관적 주장 또는 티어 선언으로 시작

예:

"단종은 정통성만 보면 SSS 티어였습니다"

"하지만 조선 서버 메타에서는 살아남기 어려운 구조였습니다"


# Stage 2 Server Meta Setup

당시 권력 구조 설명

포함 요소:

왕권 구조
신권 구조
군사 구조
외척 구조
파벌 관계
외부 위협

목적:

개인의 실패가 아니라 서버 구조 문제로 설명


# Stage 3 Base Stats Analysis

다음 항목 중 최소 5개 사용

legitimacy
political_power
military_power
support
intelligence
mobility
defense
resource_control
survivability

항상 비교 기반 설명 사용


# Stage 4 Special Abilities

캐릭터 고유 능력 설명

예:

혈통 정통성
외척 네트워크
군권 장악력
명분 생산 능력
정보 장악력
기습 실행 능력


# Stage 5 Matchups

주요 경쟁자와의 관계 설명

항상 다음 순서 유지

거리 advantage
공격력 비교
방어력 비교
기동성 비교
지원 세력 비교
카운터 여부 판단


# Stage 6 Meta Shift Analysis

환경 변화 설명

예:

왕위 계승 변화
쿠데타 발생
파벌 재편
외침 발생
제도 개편


# Stage 7 Tier Placement Verdict

최종 평가 구조

강점 요약
약점 요약
메타 적합성 평가
최종 티어 선언
