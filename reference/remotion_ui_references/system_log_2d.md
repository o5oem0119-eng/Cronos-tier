# Remotion Reference — 2D System Log

Create a flat system log overlay UI.

Resolution: 1920x1080
FPS: 30

## Visual Style

terminal style flat panel

background:

dark navy

text:

terminal green

font:

JetBrains Mono

no glow
no gradients
no blur

## Layout

top-left aligned panel

monospace text

line-by-line typing animation

example:

[SYSTEM LOG]

Player: Danjong
Status: Poisoned
Authority: Revoked

## Animation

typing effect

cursor blink allowed

panel fade-in allowed

## Component Name

<SystemLog />
