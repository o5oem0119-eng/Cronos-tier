# 🎮 TierZoo Styling Guide (업무 매뉴얼)

이 스킬은 모든 에이전트(전략가, 작가, 디렉터)가 동일한 톤앤매너와 분석 기준을 유지하도록 돕는 지침서입니다.

## 1. 📖 용어 치환 사전 (Glossary)

| 역사적 용어 | 게임적 치환 용어 | 비고 |
| :--- | :--- | :--- |
| **인물** | 플레이어 / 캐릭터 / 빌드 (Build) | 구체적인 특성에 따라 '빌드'로 표현 |
| **사건** | 서버 이벤트 / 밸런스 패치 / 퀘스트 | 시대적 변화는 '패치 내역' |
| **죽음** | 계정 삭제 / 로그아웃 / 서비스 종료 | 영구적인 탈락 의미 |
| **권력** | STR(물리력) / 서버 관리자 권한 | 직접적인 통제권 |
| **지략** | INT(지능 스탯) / 테크트리 / 전략 스크립트 | 지능적인 해결 방식 |
| **명망/정통성** | CHA(카리스마) / 적통성 버프 / 서버 인증 키 | 시스템적 정당성 |
| **유배** | 맵 격리 / 특정 바이옴 감금 / 서버 추방 | 활동 영역 제한 |
| **조력자** | 가디언 NPC / 파티원 / 서포터 | 협력 관계 |
| **적** | 메타 브레이커 / 카운터 픽 / 하드 카운터 | 상성상 우위 존재 |
| **시대적 한계** | 시스템 제약 / 엔진 사양 / 미구현 기능 | 어쩔 수 없는 환경적 요인 |
| **압도적 능력** | 오버파워(OP) / 오버튠(Overtuned) / 밸붕 | 밸런스를 파괴하는 사기적 성능 |

## 2. 🏛 기본 서사 구조 (Narrative Structure)

1.  **인트로 (The Meta Explanation)**: 
    - 해당 시대가 어떤 '메타'인지 정의. 광고 구간 없이 바로 본론으로 진입.
    - 예: "조선 서버는 정통성이 없으면 바로 강퇴당하는 하드코어 적통성 메타입니다."
2.  **캐릭터 스탯 분석 (Character Stat Profile)**: 
    - 주요 플레이어의 스탯(STR, INT, LUK, CHA, HP) 분포 분석.
    - 부친/조부로부터 물려받은 '상속 아이템'이나 '패시브 버프' 설명.
3.  **주요 퀘스트 및 상성 분석 (Matchup Spread & Quest)**: 
    - 역사적 사건을 게임의 보스 레이드나 PVP 상성 대결로 묘사.
    - 특정 인물이 패배한 이유를 '하드 카운터'나 '컨트롤 미스'로 분석.
4.  **최종 티어 결정 (Final Tiering)**: 
    - S-Tier부터 F-Tier까지 해당 인물의 효율성(Viability)을 확정.
    - 현재 메타에서 해당 빌드가 얼마나 효과적이었는지 최종 평가.

## 3. 🎯 핵심 작법 원칙

- **객관적 논리**: 단순히 웃기기 위한 농담이 아니라, 역사적 인과관계를 게임 메커니즘(상성, 스탯 부족 등)으로 설득력 있게 치환하십시오.
- **감정 배제**: 역사의 비극을 서술할 때도 TierZoo처럼 냉소적이고 객관적인 '리뷰어'의 시선을 유지하십시오.
- **광고 배제**: 실제 TierZoo 영상의 Sponsorship 구간은 대본 작성 시 생략하고 본론의 정보 밀도를 높입니다.

> [!TIP]
> "왜 죽었나?"라는 질문에 "상대방의 STR 스탯이 압도적이었고, 본인의 CHA 버프가 시스템적으로 작동하지 않는 바이옴이었기 때문"과 같이 시스템적인 답변을 내놓는 것이 성공적인 스타일링입니다.
