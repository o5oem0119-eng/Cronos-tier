# Chronos TierZoo Matchup Logic

## Purpose

이 문서는 역사 인물 또는 세력 간 권력 경쟁을 PvP matchup 구조로 분석하기 위한 규칙이다.

모든 matchup 분석은 반드시 아래 순서를 따른다.


# Step 1 Distance Advantage

먼저 누가 먼저 행동 가능한 위치에 있었는지 분석한다.

예

왕권 접근 가능 여부
군권 접근 가능 여부
정보 접근 가능 여부


# Step 2 Power Comparison

다음 요소 비교

군사력
정치력
동원 능력
실행 속도

누가 initiative 확보 가능한지 판단한다.


# Step 3 Defense Comparison

다음 요소 비교

숙청 방어 가능 여부
암살 방어 가능 여부
정치적 보호막 존재 여부


# Step 4 Mobility Advantage

다음 요소 비교

행동 자유도
이동 가능성
결정 속도
위기 대응 능력


# Step 5 Support Comparison

다음 요소 비교

신하 세력
군부 지지
외척 지지
민심 지지


# Step 6 Counter Detection

다음 요소 확인

명분 counter 존재 여부
제도 counter 존재 여부
군사 counter 존재 여부
파벌 counter 존재 여부


# Step 7 Matchup Result

결과는 반드시 다음 중 하나로 선언

dominant matchup
even matchup
disadvantage matchup


# Writing Rule

모든 matchup 설명은 반드시 비교 기반으로 작성한다.

예

"정통성에서는 단종이 우위였습니다."

"하지만 군권에서는 수양대군이 압도적인 advantage를 가지고 있었습니다."
