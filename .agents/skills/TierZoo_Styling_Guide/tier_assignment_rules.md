# Chronos Tier Assignment Rules

## Purpose

이 문서는 역사 인물 또는 세력의 최종 Tier 평가 기준을 정의한다.

Tier 평가는 반드시 gameplay viability 기준으로 수행한다.


# Evaluation Criteria

다음 요소를 종합 평가한다

legitimacy
political_power
military_power
support
resource_control
counter_vulnerability
meta_fit
survivability


# S Tier

조건

모든 matchup에서 안정적 우위

정통성 또는 군권 중 최소 하나 S 이상

counter 거의 없음

meta compatibility 매우 높음


# A Tier

조건

대부분 matchup에서 우위

일부 counter 존재

환경 의존성 낮음


# B Tier

조건

상황 의존적 강점 존재

특정 환경에서 강력

meta dependency 존재


# C Tier

조건

특정 niche 상황에서만 강함

support dependency 높음


# D Tier

조건

구조적 약점 존재

counter 다수 존재


# F Tier

조건

권력 구조상 생존 불가능

counter unavoidable

server meta mismatch


# Final Verdict Rule

Verdict는 반드시 다음 순서로 작성

1 강점 요약
2 약점 요약
3 meta compatibility 평가
4 최종 tier 선언


# Example Verdict

"정통성은 매우 높았지만"

"군권과 정치력이 거의 없었기 때문에"

"당시 조선 서버 메타에서는 생존할 수 없는 구조였습니다"

"그래서 최종 평가는 F tier입니다"
