# Chronos TierZoo Metaphor Dictionary

## Purpose

이 문서는 역사 콘텐츠를 TierZoo 스타일의 게임 메타 언어로 변환하기 위한 비유 사전이다.

Chronos Engine은 아래 대응 규칙을 사용하여 역사 설명을 gameplay 설명으로 변환한다.


# Core Mapping Rules

역사 인물 = player build
왕 = server admin build
왕세자 = future admin build
섭정 = temporary admin override

세력 = faction
신하 집단 = guild
외척 = support faction
군부 = combat guild

왕조 = server
국가 = server instance
제국 = multi-server power

시대 변화 = expansion patch
제도 변화 = system update
정책 변화 = balance patch

쿠데타 = PvP encounter
숙청 = hard counter event
암살 = stealth kill mechanic

전쟁 = cross-server PvP
내전 = faction PvP

정통성 = passive legitimacy buff
명분 = narrative advantage modifier

민심 = support meter
여론 = popularity stat

정보망 = vision control
첩보 조직 = stealth detection system

군권 장악 = combat authority unlock

왕위 계승 = admin privilege transfer


# Meta-Level Mapping

환경 변화 = meta shift
외침 발생 = external server invasion event
파벌 재편 = faction restructuring patch

권력 공백 = server instability window
섭정 체제 = admin proxy mode

정치 개혁 = system rebalance
군제 개혁 = combat system overhaul


# Collapse Mapping

몰락 = build failure
유배 = map exile state
폐위 = admin privilege revoked
처형 = permanent character removal
왕조 멸망 = server shutdown event
