# Remotion Reference — 2D Stat Card

Create a flat 2D stat card UI component.

Resolution: 1920x1080
FPS: 30

## Visual Style

strictly lineless vector art
flat cel-shading
no outlines
no gradients
no glow
no textures
no shadows

## Layout

center hexagon radar chart

left:

character name
class label

right:

tier badge

bottom:

passive traits
debuff traits

## Stat Schema

use:

legitimacy
political_power
military_power
intelligence
mobility
defense
support
resource_control
survivability

## Animation

hexagon grows from center

stat polygon fills sequentially

tier badge pops

labels slide upward

## Component Name

<StatCard />
