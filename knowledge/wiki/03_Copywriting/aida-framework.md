# 📋 [AIDA-Framework] 유튜브 자동화 LLM을 위한 스크립트 작성 가이드

이 문서는 Attention(주의), Interest(흥미), Desire(욕구), Action(행동)의 4단계를 통해 시청자의 심리 여정을 매끄럽게 유도하는 검증된 마케팅 프레임워크를 유튜브 스크립트에 적용하기 위한 지침입니다.

---

## 1. AIDA 구조 및 유튜브 적용 전략

AIDA는 시청자의 초점을 **'그들(시청자) → 우리(크리에이터) → 그들 → 우리'** 순으로 전환하며 몰입을 완성합니다.

### **🅰️ Attention (주의 / 후킹) [0~10초]**
* **초점:** 시청자 (Them)
* **전략:** 시청자의 고통(Pain point)을 찌르거나 파격적인 결과물을 먼저 약속하십시오.
* **핵심:** 3초 안에 스크롤을 멈추게 하는 시각적/청각적 패턴 깨기(Pattern interrupt).

### **🇮 Interest (흥미 / 신뢰 구축) [도입부 직후]**
* **초점:** 우리 (Us)
* **전략:** 시청자의 문제에 깊이 공감하고 있음을 보여주며 전문가적 통계나 신뢰할 만한 근거를 제시하십시오.
* **핵심:** 왜 이 영상을 끝까지 봐야 하는지에 대한 타당한 문맥(Context) 제공.

### **🇩 Desire (욕구 / 혜택 제시) [본론]**
* **초점:** 시청자 (Them)
* **전략:** 기능(Feature)의 나열이 아닌, 해결책을 통해 시청자의 삶이 어떻게 변하는지(Benefits)를 시각화하십시오.
* **핵심:** 실제 사례(Case Study)나 시연을 통해 '변화된 모습'에 대한 욕구를 자극.

### **🅰️ Action (행동 / 콜투액션) [마지막 10~20초]**
* **초점:** 우리 (Us)
* **전략:** 헷갈리지 않게 **단 하나의 명확한 행동**만 지시하십시오.
* **핵심:** 행동을 취했을 때 얻게 될 가치를 재상기시키거나 긴급성(Urgency)을 부여.

## 2. 🤖 LLM 지시사항 (System Prompt / Instructions)

자동화 LLM은 스크립트 작성 시 다음 지침을 엄격히 준수해야 합니다.

1.  **구어체 및 호흡 조정 (Write for the ear):**
    * 짧고 간결한 문장(15~20단어 내외)을 사용하며, 친구와 대화하듯 축약어와 구어체를 활용하라.
    * 1분당 130~150단어의 속도를 기준으로 분량을 조절하라.
2.  **후킹 최우선 법칙 (Hook Priority):**
    * 본문을 쓰기 전, 강력한 후킹 아이디어를 3가지 이상 먼저 제안하라.
    * "안녕하세요", "오늘 영상에서는" 같은 무의미한 인사말로 시작하는 것을 절대 금지한다.
3.  **2단 포맷 활용 (Two-Column Format):**
    * `[B-roll]`, `[텍스트 오버레이]` 등 시각 지시문과 오디오 대사를 명확히 구분하라.
4.  **시청자 초점 전환 규칙 준수:**
    * A(그들) → I(우리) → D(그들) → A(우리)의 초점 이동을 스크립트 구조에 반영하라.
    * 특히 Desire 단계에서 시청자가 얻게 될 감정적 만족감을 반드시 묘사하라.
5.  **단일 CTA 원칙 (Single CTA Rule):**
    * 영상 마지막에는 오직 '가장 중요한 한 가지 행동'만 명확한 동사로 지시하라. (예: "구독하세요" 대신 "지금 아래 링크를 클릭해 비법을 확인하세요")
