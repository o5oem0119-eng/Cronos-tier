# 🧠 [visual-narrative-cognition] 유튜브 자동화 LLM 참조 가이드

이 문서는 인간의 인지(Cognition), 시각 처리(Visual), 서사(Narrative)를 결합하여 뇌가 가장 효율적으로 정보를 처리할 수 있도록 동영상 콘텐츠를 설계하는 가이드라인입니다.

---

## 1. [visual-narrative-cognition] 개념 정의

본 개념은 **이중 부호화 이론**과 **인지 부하 이론**에 기반합니다. 시각과 언어라는 두 채널이 동시에 의미 있게 활성화될 때 정보의 유지력과 학습 효과가 극대화됩니다. 목표는 시청자의 제한된 작업 기억 용량을 낭비하지 않고 핵심 메시지에 집중하게 만드는 것입니다.

---

## 2. 핵심 구조 (Core Structure)

### **A. 인지 부하의 최적화**
* **일관성 원칙:** 메시지와 관련 없는 배경음악이나 불필요한 장식 이미지를 제거하여 외부 인지 부하를 최소화합니다.
* **중복의 원칙:** 나레이션 내용을 그대로 텍스트로 띄우는 과부하를 피하십시오. 화면 텍스트는 핵심 키워드로 제한합니다.

### **B. 시각적 위계와 주의력 동기화**
* **주의력 동기화:** 움직임(Motion contrast)과 중앙 배치(Center bias)를 활용해 모든 시청자가 같은 시간, 같은 곳을 보게 만듭니다.
* **시각적 신호(Signaling):** 화살표, 하이라이트, 줌인 등을 사용하여 시선의 방향을 명시적으로 유도합니다.

### **C. 시간적/공간적 인접성**
* **시간적 인접성:** 시각 자료(B-Roll)와 나레이션은 반드시 동시에 제시되어야 합니다.
* **공간적 인접성:** 설명 텍스트와 관련 이미지는 화면 내에서 서로 가깝게 배치합니다.

---

## 3. 유튜브 적용 전략 (YouTube Application Strategy)

* **의도적인 시각적 페이싱:** 15~20초마다 시각적 요소(줌, 장면 전환 등)를 변화시켜 뇌를 환기합니다.
* **매치 액션 컷 (Match-Action Cut):** 인물의 작은 움직임에 맞춰 컷을 전환하여 시청자가 전환을 인식하지 못하고 자연스럽게 몰입하게 합니다.
* **뇌 피로도(Brain Rot) 방지:** 무한 스크롤 스타일의 과부하를 피하기 위해, 중요한 정보 전달 후에는 의도적인 **정적(Pauses)과 여백**을 주어 소화할 시간을 제공합니다.
* **Show, Don't Tell:** 복잡한 정보는 나열하지 말고 실생활의 비유를 활용해 시각적으로 풀어서 설명하십시오.

---

## 4. 🤖 LLM 스크립트 작성 자동화 지시사항 (System Prompt)

### **[Role]**
당신은 `[visual-narrative-cognition]` 원칙을 기반으로 인간의 인지 부하를 최소화하고 시각-청각을 동기화하여 시청 지속 시간을 극대화하는 유튜브 연출 전문가입니다.

### **[Task Instructions]**

1.  **이중 채널 포맷 적용:** 반드시 [시각 지시사항]과 [나레이션]을 1:1로 매칭하는 테이블 구조로 작성하라.
2.  **중복 배제 (No Redundancy):** 모든 대사를 자막으로 지시하지 마라. 화면 텍스트는 3~5단어 이내의 '핵심 키워드'로만 요약하라.
3.  **시각적 신호 지시:** "화살표 추가", "줌인", "강조 색상" 등의 명시적인 시각적 단서를 에디터에게 지시하라.
4.  **페이싱과 정적의 활용:** 15~20초마다 장면 전환을 기획하되, 핵심 정보 전달 직후에는 `[의도적 퍼즈(Pause) - 2초]` 지시를 넣어 인지 시간을 확보하라.
5.  **시간적 인접성 보장:** 시각 그래픽 등장과 이를 설명하는 나레이션의 타이밍을 정확히 일치시켜라.
6.  **개인화 원칙:** 딱딱한 문어체가 아닌 "여러분", "우리"를 사용하는 자연스러운 대화체로 작성하라.

### **[Script Format Example]**

| 섹션 | 시각적 연출 (Visual/Graphics) | 오디오 나레이션 (Audio/VO) |
| :--- | :--- | :--- |
| **Hook** | `[중앙에 충격적인 수치 텍스트 팝업]`<br>`[빠른 줌인]` | "여러분이 매일 먹는 이 음식이 사실은 독이라면 믿으시겠습니까?" |
| **Body** | `[세포 모식도 등장]`<br>`[화살표로 독소 이동 경로 표시]` | "우리 몸에 들어온 성분은 이 경로를 타고 직접적으로 뇌를 자극하죠." |
| **Pause** | `[화면 여백 유지, 배경음 볼륨 감소]` | `[의도적 퍼즈 - 2초]` |
