# 🏗️ Blueprint: 단종 에피소드 영상 대본 설계 (Chronos Engine)

본 설계도는 `단종_비극_게임분석.md`를 기반으로, 역사적 사건을 게임 패치노트 및 메타 분석 스타일의 영상 대본으로 변환하기 위한 구조 정의서입니다.

## 1. 영상 서사 구조 (Narrative Layout)

### [Phase 1: 인트로 - 조선 서버 접속]
- **비주얼:** 픽셀 아트 기반의 조선 궁궐 배경 + 'Loading Joseon Server...' 자막.
- **나레이션:** "조선이라는 국가 시뮬레이션 서버에서 가장 완벽한 '적통성' 스탯을 가진 캐릭터가 탄생했습니다. 하지만 왜 그는 'Game Over'를 당했을까요?"
- **게임 요소:** '적통성(Legitimacy)' 시스템 및 서버 규칙(유교 종법) 설명.

### [Phase 2: 캐릭터 프로필 - SSS급 마스터피스]
- **비주얼:** 단종의 캐릭터 카드 UI (스탯 바: Legitimacy SSS, Charisma S, Intelligence A+).
- **나레이션:** "문종의 유전자를 상속받은 하이엔드 캐릭터. 외모와 지능, 혈통까지 완벽한 SSS급 가차 결과물입니다."
- **게임 요소:** 아이템 상속(문종의 지능), 패시브 스킬(복종 유도) 분석.

### [Phase 3: 시스템 결함 - 가디언 글리치]
- **비주얼:** 캐릭터 카드 주변의 'Guardian' 슬롯이 비어있음을 빨간색 경고등으로 표시.
- **나레이션:** "하지만 이 완벽한 캐릭터에게 치명적인 버그가 발생합니다. 바로 그를 보호할 '가디언 NPC'들이 모두 서버에서 삭제된 것이죠."
- **게임 요소:** 수렴청정(Regency) 보호 모드 미작동, 고립(Isolated) 디버프 발생.

### [Phase 4: 메타 브레이커의 습격 - 계유정난]
- **비주얼:** 수양대군의 'Power Creep' 빌드 분석 UI (무력 및 야심 스탯 폭발).
- **나레이션:** "정통성 메타를 무시하고 물리적 파괴력으로 서버를 해킹한 유저, 수양대군이 등장합니다. 한명회의 '살생부' 스크립트가 실행됩니다."
- **게임 요소:** 다이렉트 킬(Direct Kill) 전술, 관리자 권한(Admin Privileges) 탈취.

### [Phase 5: 엔딩 - 강제 로그아웃]
- **비주얼:** 영월 청령포 지형의 '바이옴(Biome)' 격리 맵 시각화.
- **나레이션:** "결국 격리 맵으로 유배된 단종. 17세라는 나이에 최종 삭제 처리되며 서버 역사상 가장 억울한 밴(Ban)으로 기록됩니다."
- **게임 요소:** 격리 맵(청령포), 최종 밴(사약/질식사설).

## 2. 데이터 구조 (Data Assets)

### [Script Metadata Schema]
- `scene_id`: 장면 식별자
- `narration`: 성우 지문 (Gamer Lingo 포함)
- `overlay_ui`: 화면에 나타날 게임 UI 요소 (스탯 바, 경고창 등)
- `bgm_intent`: 장면의 분위기에 따른 음악 스타일

### [Character Stat Mapping]
- `legitimacy`: 0 ~ 100 (단종: 100, 수양: 20)
- `ambition`: 0 ~ 100 (단종: 10, 수양: 100)
- `power`: 0 ~ 100 (단종: 5, 수양: 95)

---

## 3. 파일 구조 제안 (File Structure)

- `data/`
  - `danjong_script.json`: 실제 생성된 대본 데이터
  - `stats_master.json`: 모든 등장 인물의 스탯 테이블
- `directives/`
  - `script_rules.md`: 말투 및 용어 가이드
- `src/remotion/`
  - `CharacterCard.tsx`: 스탯 바 애니메이션 컴포넌트
  - `SystemLog.tsx`: 서버 로그 출력 컴포넌트

---

## 4. 제어 지침 (Control Rules)
- **용어 통일:** 반드시 '서버', '패치', '삭제(밴)', '스탯', '빌드' 등의 게임 용어를 역사적 용어와 병기하거나 대체할 것.
- **톤앤매너:** Tier Zoo의 시니컬하면서도 전문적인 정보 전달 톤 유지.
- **비주얼 무드:** '시스템 관리 대시보드'와 '고전 도트 아트'의 결합.
