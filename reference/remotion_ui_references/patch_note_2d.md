# Remotion Reference — 2D Patch Note Panel

Create a flat patch note announcement panel.

Resolution: 1920x1080
FPS: 30

## Visual Style

server update window style

flat rectangle panel

dark overlay background

light text

no gradients
no glow
no shadows

## Layout

title:

PATCH NOTE

subtitle:

server update event

body:

bullet list changes

example:

PATCH NOTE

Guardian System Removed

Military Control Shifted

Faction Balance Changed

## Animation

panel slides from top

text fades sequentially

## Component Name

<PatchNote />
