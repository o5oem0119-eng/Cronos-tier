# I.hezz.co 브랜드 비주얼 시스템 가이드라인 (최종 확정)

## 1. 핵심 컬러 팔레트 (Core Palette)
브랜드의 신뢰감과 친근함을 동시에 전달하는 색상 조합입니다.

*   **Primary Color (메인):** `#1A237E` (딥 네이비/인디고 계열) - 신뢰, 전문성, 깊이감을 표현합니다.
*   **Accent Color (강조):** `#64FFDA` (밝은 민트/아쿠아 계열) - 활력, 친근함, 주목도를 높이는 데 사용합니다.
*   **Neutral Colors (보조):**
    *   Background: `#FFFFFF` (순백색) - 가독성과 깔끔함을 극대화합니다.
    *   Text/Primary Contrast: `#1A237E` 또는 블랙 계열

## 2. 타이포그래피 시스템 (Typography System)
가독성과 친근함을 동시에 확보하는 조합입니다.

*   **Headline Font (제목):** Pretendard Bold 또는 Noto Sans Bold 계열 - 강력하고 현대적인 인상을 줍니다.
*   **Body Font (본문):** Pretendard Regular 또는 Noto Sans Regular - 깔끔하고 읽기 쉬운 본문을 제공합니다.

## 3. 레이아웃 및 구성 규칙 (Layout & Composition Rules)

### A. 유튜브 썸네일 표준 레이아웃
1.  **배경:** `#1A237E` 또는 어두운 계열의 배경을 사용하여 전문성을 확보합니다.
2.  **강조 요소:** 강조 컬러(`#64FFDA`)를 활용하여 핵심 메시지나 숫자를 배치합니다.
3.  **텍스트 배치:** 중앙 및 우측에 주요 후크(Hook) 문구를 배치하며, 폰트 크기는 시각적 위계를 명확히 구분해야 합니다.
4.  **인물/아이콘:** 필요시 얼굴 또는 관련 아이콘을 활용하여 친근한 느낌을 더합니다.

### B. 인스타그램 카드뉴스 표준 레이아웃 (3~5장 기준)
1.  **일관성:** 모든 슬라이드는 동일한 폰트 스타일(Headline/Body)과 컬러 팔레트를 유지해야 합니다.
2.  **시각적 흐름:** 각 페이지는 하나의 명확한 메시지(One Message Per Slide)를 전달하도록 구성합니다.
3.  **색상 사용:** 배경은 `#FFFFFF`를 기본으로 하며, 강조 컬러(`#64FFDA`)를 포인트로 사용하여 시선을 유도합니다.
4.  **여백:** 충분한 여백(White Space)을 확보하여 정보가 압축되지 않고 친근하게 보이도록 합니다.

## 4. 디자인 적용 원칙 (Application Principles)
*   **친근함 우선:** 전문적인 내용을 전달하더라도 딱딱하지 않고, 독자가 '나와 대화하는' 느낌을 주어야 합니다.
*   **명확성 최우선:** 복잡한 정보는 시각적 계층 구조(Hierarchy)를 통해 즉시 이해 가능하게 만듭니다.
*   **컬러 사용 제한:** 메인 컬러(`#1A237E`)와 강조 컬러(`#64FFDA`)의 비율을 7:3 또는 6:4 정도로 유지하여 브랜드 일관성을 확보합니다.
